## 🖼️ 유튜브 썸네일 시안 (Mockup 1/3)

**주제:** 금융/투자 시스템의 파국적 붕괴 (The Financial Catastrophe)
**목표:** 극단적인 손실의 시각적 긴장감과 시스템의 취약점을 강조하여 클릭을 유도.

### 1. 시각 시스템 적용 상세

*   **배경:** 딥 블랙 (Deep Black) 또는 매우 짙은 차콜 그레이.
*   **핵심 강조 색상:**
    *   **경고/위험 (Red):** 시스템 붕괴 지점, 손실 그래프, 결정적인 실패 지점에 사용.
    *   **분석/경고 (Cyan):** 시스템의 취약점, 분석 텍스트, 핵심 데이터 포인트에 사용.
*   **타이포그래피:** 고딕 계열의 굵고 날카로운 산세리프체 (예: Impact, Bebas Neue 또는 유사한 굵은 고딕체).

### 2. 레이아웃 및 구성 (Aspect Ratio: 16:9)

| 영역 | 내용 | 색상/효과 | 설명 |
| :--- | :--- | :--- | :--- |
| **배경** | 전체 화면 | `#0A0A0A` (거의 블랙) | Dark Aesthetic 극대화. |
| **중앙 이미지** | 추상적인 데이터 시각화 (파국적 붕괴 그래프) | `#FF0000` (Red) 선과 `#00FFFF` (Cyan) 선의 대조 | 극단적인 대비를 통해 시각적 충격 부여. |
| **텍스트 (메인 헤드라인)** | "시스템 붕괴 시퀀스" | `#00FFFF` (Cyan) | 가장 중요한 정보를 Cyan으로 강조하여 '분석'의 느낌 부여. 폰트 크기 최대화. |
| **텍스트 (서브 캡션)** | "실시간 손실 그래프 해부" | `#FFFFFF` (White) | 보조 정보를 명확하게 전달. |
| **경고 마크** | 중앙 또는 우측 상단 | `#FF0000` (Red) 원 또는 삼각형 | 시스템의 위험성을 즉각적으로 알림. |

### 3. 디자인 결과 (텍스트 기반 시안)

**[썸네일 시안 명: Catastrophe_Sequence_V1]**

*   **배경:** 딥 블랙 (`#0A0A0A`)
*   **중앙:** 거대한 붉은색 (`#FF0000`)으로 그려진 급락하는 그래프 라인들이 화면을 가로지름.
*   **텍스트 배치:**
    *   **상단 중앙 (Cyan):** **SYSTEM FAILURE** (매우 굵고 날카로운 폰트)
    *   **하단 (White):** The Unseen Collapse: Real Loss Data (Cyan 강조)
*   **효과:** 전체적으로 날카로운 선과 짙은 명암 대비로 긴장감을 극대화합니다.
